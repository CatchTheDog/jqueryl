定位测试

div1 static
div2 relative top:50px; left:100px;
div4 relative; top: 50px; left:100px;
div5 absolute; top: -50px; left:100px;
div6 static
div3 static
div7 absolute;top: 50px;left: 100px;

定位总结

  • position有几个重要的值:static relative absolute
  • static:默认的，正常文档流元素的position属性所具有的值
  • relative：当元素position值为relative时，表示该元素相对于其在正常文档流中的位置发生偏移，具体偏移距离根据top,left,right,bottom决定
  • absolute:当元素position值为absolute时，表示该元素将脱离文档流，且相对于其包含块（其父类块中position值不为static的元素）发生偏移，具体偏移距离根据top,left,right,bottom决定
  • z-index 属性只在position为relative,absolute,fixed的元素上有效，值比较大的元素会出现在值比较小的元素上面
